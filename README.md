### Vite + Svelte + TS + ESlint + Prettier

Svelte Typescript Vite template with Eslint and Prettier configured

Vite
```bash
// Vite
npm run dev
npm run build
npm run preview
```

ESlint
```bash
// ESlint
npm run lint
npm run lint:fix
```

Prettier
```bash
// Prettier
npm run format
npm run format:fix
```
